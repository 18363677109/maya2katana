# maya2katana
Easily copy shading nodes from Maya to Katana

Currently supported nodes (Arnold with alShaders):
- aiAmbientOcclusion
- aiImage
- aiNoise
- aiUserDataColor
- aiUserDataFloat
- alCellNoise
- alCombineColor
- alCombineFloat
- alCurvature
- alFlake
- alFlowNoise
- alFractal
- alHair
- alInputScalar
- alInputVector
- alJitterColor
- alLayer
- alLayerColor
- alLayerFloat
- alRemapColor
- alRemapFloat
- alSurface
- alSwitchColor
- alSwitchFloat
- alTriplanar
- blendColors
- bump2d
- clamp
- luminance
- ramp
- samplerInfo
